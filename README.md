# 🛒 Blinkit Sales Dashboard - Power BI

## 📌 Project Overview

This project presents an interactive **Power BI dashboard** developed to analyze Blinkit's sales performance and customer insights. The dashboard helps identify business trends, monitor key performance indicators (KPIs), and support data-driven decision-making.

---

## 🎯 Objectives

- Analyze overall sales performance.
- Track important business KPIs.
- Understand customer preferences.
- Compare sales across different outlet locations.
- Identify patterns based on product characteristics.

---

## 📊 Dashboard Features

### Key Performance Indicators (KPIs)
- 💰 Total Sales
- 📦 Number of Items Sold
- ⭐ Average Rating
- 📈 Average Sales

### Visualizations Included
- Sales distribution by **Item Fat Content**
- Comparison of sales across **Outlet Location Types**
- Interactive slicers for filtering insights
- Dynamic KPI cards for quick analysis
- Additional charts and visuals for business exploration

---

## 🛠️ Tools & Technologies Used

- **Power BI Desktop**
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Interactive Visualizations
